Accueil / Paiement

Orange Money

Effectuez votre paiement via le numéro suivant :

+237 6 99 00 00 00
  1. Composez #150# sur votre téléphone
  2. Sélectionnez "Transfert d'argent"
  3. Entrez le numéro ci-dessus
  4. Indiquez le montant exact de votre commande
  5. Confirmez avec votre code secret Orange Money
  6. Envoyez la capture de confirmation par WhatsApp ou email

MTN Mobile Money

Effectuez votre paiement via le numéro suivant :

+237 6 77 00 00 00
  1. Composez *126# sur votre téléphone
  2. Sélectionnez "Transférer de l'argent"
  3. Entrez le numéro ci-dessus
  4. Indiquez le montant exact de votre commande
  5. Confirmez avec votre code secret MoMo
  6. Envoyez la capture de confirmation par WhatsApp ou email

Instructions importantes

1. Effectuez le paiement du montant exact indiqué dans le récapitulatif de votre commande.

2. Envoyez une capture d'écran de la confirmation de transaction à l'adresse email de l'administrateur : nkamambararayan@gmail.com ou par WhatsApp.

3. Votre commande passera automatiquement au statut "En cours" dès validation du paiement par notre équipe.

4. Votre photo retouchée vous sera livrée dans le délai que vous avez sélectionné lors de la commande.

Aucune information bancaire n'est demandée sur ce site. Le paiement s'effectue entièrement de façon manuelle et sécurisée via votre application Mobile Money.
